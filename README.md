# MOTOR_CONTROL
A Motor Control Application with Qt 5.x C++, qcustomplot, board stm32f407 on UbunTu. 
FW folder include: FW of stm32f1 (FW) and FW of stm32f407 (FW407). This Firmware was referenced from my teacher.
